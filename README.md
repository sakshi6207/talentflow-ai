# TalentFlow AI

TalentFlow AI is a modern Applicant Tracking System (ATS) that helps companies streamline recruitment through AI-powered resume analysis, candidate-job matching, interview management, and hiring analytics.

## Features

- Role-Based Access Control (Admin, HR, Interviewer, Candidate)
- Job Posting & Application Management
- Resume Upload & Parsing
- AI-Powered Candidate Matching
- Interview Scheduling & Feedback
- Hiring Pipeline Tracking
- Offer Letter Generation
- Analytics Dashboard
- Email & In-App Notifications

## Project Goal

Build a production-grade HR Tech platform inspired by modern ATS solutions like Greenhouse, Lever, and Zoho Recruit while leveraging AI to improve hiring efficiency and candidate evaluation.
